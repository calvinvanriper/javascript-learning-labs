## üìè Longest Word Length Lab

This lab finds the length of the longest word in a sentence. The program splits the sentence into words, loops through each word, and tracks the largest word length found.

### Example Behavior

```

‚Üí 5

findLongestWordLength("JavaScript is powerful")
‚Üí 10
```

### Concepts Practiced

- Iterating through a list of values using a loop
- Tracking a running result (maximum value)
- Working with word lengths using `.length`
- Breaking a string into words using `trim()` and `split(" ")`
